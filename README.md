# PDF Context Search

A Tauri-based desktop application for searching and highlighting text across multiple PDFs with Zotero integration.

## Quick Start

```bash
# Install dependencies
npm install

# Run in development mode
npm run tauri dev

# Build for production
npm run tauri build
```

## Usage

1. Click "Browse" to select a PDF directory
2. Enter search terms (supports regex and multi-word phrases)
3. Click "Search" to find matches across all PDFs
4. View results with highlighted context, pin important results, and export to Markdown
5. Optional: Enable Zotero mode for citation metadata and copy-to-clipboard formatting
